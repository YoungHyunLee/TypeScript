

# Basic Types


## Boolean


아래와 같이 정의할 수 있어.

```ts
var isDone: boolean = false;
```


## Number


TS에서 모든 숫자들은 부동 소수점값을 가져.
부동 소수점 수는 타입으로 `number`를 가지지.


```ts
var height: number = 6;
```


## String


문자열 데이터는 `'`이나 `"`으로 주위를 감싸서 표현할 수 있어.

```ts
var name: string = "bob";
name = 'smith';
```
